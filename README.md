# Guild Navigator 🧭

Tree-based code navigation for VS Code. Go back, go forward, choose your path.

## Features

- Navigate your code history as a tree
- Go back and forward through your navigation history
- When multiple forward paths exist, choose which branch to follow
- Tracks navigation across files

## Keyboard Shortcuts

- `Cmd+Alt+[` — Go Back
- `Cmd+Alt+]` — Go Forward

## Commands

- `Guild Navigator: Go Back`
- `Guild Navigator: Go Forward`
- `Guild Navigator: Clear History`
